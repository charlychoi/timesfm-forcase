## 1. 제품 목적

Google TimesFM 2.5 Zero-Shot 시계열 예측 모델을 활용해  
**암호화폐(BTC/ETH/XRP)** 및 **한국 주식(삼성전자/SK하이닉스/현대차/KOSPI)**의  
1/7/14/30일 가격 예측을 시각화하고,  
예측 신뢰도·변동성·백테스트 오차를 함께 제공하는 **교육용** 웹 서비스.

> ⚠️ 투자 권유 아님. 모든 예측은 과거 패턴 기반 통계적 추정.

## 3. 기능 요구사항

### 3.1 예측 기능
- [x] TimesFM 2.5 Zero-Shot 예측
- [x] 1일 / 7일 / 14일 / 30일 horizon
- [x] 10% ~ 90% 분위수 신뢰 구간
- [x] 과거 90일 + 예측 30일 통합 차트

### 3.2 신뢰도 분석
- [x] 신뢰도 점수 0~100
- [x] 신뢰도 등급: 참고 가능 / 제한적 참고 가능 / 주의 필요 / 시나리오 참고 수준
- [x] 변동성 등급: 낮음 / 보통 / 높음 / 매우 높음 (암호화폐/주식 기준 분리)
- [x] 한국어 AI 해석 문장 자동 생성

### 3.3 백테스트
- [x] 롤링 백테스트 (슬라이딩 윈도우, 최소 학습 180일)
- [x] MAPE (Mean Absolute Percentage Error)
- [x] 방향성 적중률 (Directional Accuracy)
- [x] 샘플 수 제한: 1d=30개, 7d=20개, 14d=15개, 30d=8개
- [x] 백테스트 캐시 TTL: 1시간
- [x] NaN/Inf 안전 처리 (버그픽스 v1.1-fix)

### 3.4 데이터 캐싱
- [x] 가격 캐시: 5분
- [x] 예측 캐시: 10분
- [x] 백테스트 캐시: 1시간

### 3.5 오류 처리
- [x] CoinGecko Rate Limit (429) 자동 retry (최대 3회, backoff 2초)
- [x] `safeFetchJson()`: Content-Type 검증 후 JSON 파싱
- [x] 백엔드 미연결 시 사용자 안내 배너

## 5. 기술 스택

| 레이어 | 기술 |
|---|---|
| 예측 모델 | Google TimesFM 2.5 (200M PyTorch) |
| 백엔드 | Python 3.12 + FastAPI + Uvicorn |
| 데이터 수집 | CoinGecko API (crypto) + yfinance (KR stock) |
| 프론트엔드 | 순수 HTML/CSS/JS (단일 파일) + Chart.js 4.4 |
| 백엔드 배포 | Railway (nixpacks) — 현재 비활성 |
| 프론트 배포 | Vercel (정적) — 현재 로컬 전용 |

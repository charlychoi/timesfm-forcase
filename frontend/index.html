TimesFM 2.5 · Zero-Shot
🧠
예측 모델 및 데이터 신뢰성 안내
💡 본 서비스는 예측 가격보다 예측 범위, 과거 오차, 변동성 위험을 함께 해석하는 것을 권장합니다.
이 서비스는 Google Research가 개발한 TimesFM 2.5 (Time Series Foundation Model)을 사용합니다. 1,000억 개 이상의 실세계 시계열 포인트로 사전학습된 모델로, 별도 파인튜닝 없이(Zero-Shot) 예측합니다.
🪙 암호화폐 (BTC / ETH / XRP)
데이터 출처: CoinGecko
분석 기간: 최근 365일
예측 컨텍스트: 최근 256일 종가 사용
🇰🇷 한국 주식 · 지수 (삼성전자 외 2종 + KOSPI)
데이터 출처: Yahoo Finance
분석 기간: 최근 1년 거래일
예측 컨텍스트: 최근 256일 종가 사용
💱 환율 (USD/KRW · JPY/KRW)
데이터 출처: 한국은행 ECOS API
분석 기간: 최근 2년 (ffill 보간)
주의: 주말·공휴일은 전일값으로 대체
🏢 아파트 실거래가 (서울·강남·노도강)
데이터 출처: 국토교통부 실거래가 공개시스템
분석 기간: 최근 36개월
단위: 만원/㎡ (IQR 이상치 제거 후 평균)
📊 소비자물가지수 (CPI)
데이터 출처: 한국은행 ECOS API
분석 기간: 최근 60개월
예측 주기: 월별 (1·3·6·12개월)
🤖 모델 정보
모델: TimesFM 2.5 (200M 파라미터)
예측 방식: 패턴 기반 통계 추론
신뢰 구간: 10% ~ 90% 분위수 제공
⚠️ 한계: 이 모델은 과거 가격 패턴을 학습합니다. 뉴스, 규제, 기업 실적, 거시경제 등 외부 이벤트는 직접 반영하지 않습니다. 따라서 예측 결과는 단일 가격보다 범위와 신뢰도를 함께 해석해야 합니다.
